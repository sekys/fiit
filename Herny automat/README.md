# Herny automat

Jednoducha hra postavena na freeglut. Zakladom je pozadie (herny automat) na
ktorom sa buduje 3D scena. Principom hry je trafit 3 gulicky rovnakej farby.
Hra sa ovlada jednoducho, medzernikom. Cielom bolo naucit sa pracovat s opengl
a view maticamy.

# Animacia
Hra obsahuje este druhu scenu, animaciu. Medzi animaciou a hrou sa prepina stlacenim R.
Animacia poukaziu na vyuzitie interpolacie.

#Misc
- Projekt bol napisany v Code blocks IDE.
- Projekt vyuziva papp balik od drahosp.

## Run

1. Unzip run.zip
2. Run gl_vbo.exe
